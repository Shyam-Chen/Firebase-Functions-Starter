# Puppeteer Play

## Getting Started

Follow steps to execute this boilerplate.

1. Clone this boilerplate

```bash
$ git clone --depth 1 https://github.com/Shyam-Chen/Puppeteer-Play.git <PROJECT_NAME>
$ cd <PROJECT_NAME>
```

2. Install dependencies

```bash
$ yarn install
```

3. Check code quality

```bash
$ yarn lint
```

4. Runs ui tests

```bash
$ yarn ui
```

5. Runs api tests

```bash
$ yarn api
```

6. Runs benchmarks

```bash
$ yarn bench
```
